# [Numpy](https://docs.scipy.org/doc/numpy/user/quickstart.html)库
___

### 1. 特性

         1. 快速高效的多维数组对象ndarray。
         2. 用于对数组执行元素级计算以及直接对数组执行数学运算的函数。
         3. 用于读写硬盘上基于数组的数据集的工具。
         4. 线性代数运算、傅里叶变换，以及随机数生成。
         5. 用于将C、C++、Fortran代码集成到Python的工具。
